## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on real-world conflict data from the Russia–Ukraine war.
The goal is to extract meaningful insights about **conflict patterns, fatalities, and geographic hotspots** using Python.

---

## 🎯 Objectives

* 📊 Analyze trends in conflict events over time
* 🗺️ Identify high-risk regions and hotspots
* ⚔️ Study different types of conflict events
* ☠️ Evaluate the impact using fatalities data
* 📈 Build clear and insightful visualizations

---

## 📂 Dataset

* Source: ACLED / Kaggle
* Features include:

  * Event Date
  * Event Type
  * Location (Latitude & Longitude)
  * Fatalities
  * Actors involved
